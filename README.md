# 街口捐款資料匯出工具

一個 Chrome 擴充功能，讓非營利組織後台管理員能一鍵將街口外部後台的**單次捐款交易紀錄**全數匯出為 CSV 檔案，方便在 Excel 中進行統計與對帳。

---

## 功能

- 自動偵測總頁數，逐頁爬取所有交易資料
- 匯出欄位：**扣款時間、扣款金額、專案名稱、捐款人姓名**
- 產出 UTF-8 BOM 編碼的 CSV，直接用 Excel 開啟不會亂碼
- 檔名自動帶入當天日期，例如 `街口捐款交易紀錄_20260610.csv`

---

## 安裝方式

此擴充功能尚未上架 Chrome Web Store，請用開發者模式手動安裝：

1. 下載或 clone 此 repo
2. 打開 Chrome，網址列輸入 `chrome://extensions`
3. 右上角開啟「**開發人員模式**」
4. 點選「**載入未封裝項目**」，選擇此 repo 的資料夾
5. 擴充功能安裝完成，工具列會出現圖示

---

## 使用方式

1. 登入 [街口外部後台](https://partner-admin.jkos.com)
2. 進入「**單次捐款交易紀錄**」頁面
3. 如有需要，先用搜尋條件篩選想要的資料範圍
4. 點選工具列的擴充功能圖示，按下「**開始匯出全部資料**」
5. 擴充功能會自動翻頁並收集所有資料，完成後自動下載 CSV

> ⚠️ 匯出期間請勿手動操作頁面，以免影響資料擷取。


---

## CSV 欄位說明

| 欄位 | 說明 |
|------|------|
| 扣款時間 | 交易發生的日期與時間 |
| 扣款金額 | 單筆捐款金額（新台幣） |
| 專案名稱 | 捐款所屬的募款專案 |
| 捐款人姓名 | 捐款人的姓名 |

---

## 注意事項

- 僅支援 `partner-admin.jkos.com` 網域
- 請確認已在**單次捐款交易紀錄**頁面再啟動匯出，否則會出現「無法取得頁數」的錯誤
- 資料量大時（頁數多），匯出需要一些時間，請耐心等待
